# 支付的种类说明

# 常用的支付渠道

  • 微信支付：40%
  • 支付宝：54%
  • 银联支付：0.4%

# 不同的支付场景使用不同支付方式

  • 在微信小程序的里面付款支付，需要使用小程序支付接口
  • 在微信里面打开网页进行付款，要使用JSAPI支付接口
  • 在手机浏览器上面使用微信付款，要用H5支付接口
  • 如果选用微信扫码支付，需要调用Native支付接口

# 学习路径

微信支付：小程序支付、JSAPI、H5支付、Native 支付宝：小程序支付、H5支付、Native 银联：H5支付、Native 聚合支付：微信支付、支付宝、银联

# 付款码和收款码的区别

  • 为什么商家有的时候要采用付款码收款，有的时候采用收款码收款呢？
  • 付款码可以接入商户系统，而收款码无法接入商户系统

# JSAPI支付场景

如果用户在微信内置浏览器里面，打开电商网站下单支付，这时候电商网站需要调用微信的JSAPI支付接口

# 开通微信商户平台

  • 微信商户平台是专门提供给商户管理微信支付收款的网站
  • https://pay.weixin.qq.com

# 微信商户平台的作用

  • 为商家提供微信收款和转账，并且提供交易的详细信息
  • 微信商户平台可以管理各种支付接口的开通和关闭

# 微信支付业务需要的材料

  • 除了小程序的AppID和AppSecret之外，我们还需要从微信商户平台得到商户号、数字证书和密钥
