注意点

  • 查询结果如果要显示tag，则无需指定查询tag，只需要在GROUP BY 中带上tag。
  • 如果GROUP BY 除了time 还有其它tag，则最好不要加fill(previous)，否则数据可能会错乱。
  • SELECT * FROM (SELECT …) 会将值为空的数据过滤掉。
  • GROUP BY time() 之后要确定用FIRST() 或 LAST()。
  • 如果查询的字段带有空格，则需要使用”字段名”查询。
